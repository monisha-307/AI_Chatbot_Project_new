AI Chatbot Project

A simple AI chatbot built using Python and Google Gemini API.

Features

- Interactive chatbot conversation
- Maintains conversation history
- Uses Gemini 2.5 Flash model
- Exit option to stop the chatbot

Technologies Used

- Python
- Google Gemini API

How to Run

1. Install the required package:
   pip install google-genai

2. Add your Gemini API key in the code.

3. Run the program:
   python chatbot.py
